📋 Como configurar o Supabase

1. Acesse supabase.com → Criar conta gratuita
2. Criar novo projeto → Anotar a senha do banco
3. Aguardar ~2 min → Acessar Project Settings → API
4. Copiar Project URL e anon public key
5. Colar no início do index.html nos campos indicados
6. Subir no GitHub — pronto!
