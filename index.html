Verifique se tudo está configurado corretamente para salvar na nuvem.

📋 Como configurar

1. Acesse console.firebase.google.com
2. Crie um projeto gratuito → Adicione um app Web
3. Copie o firebaseConfig e cole no início do index.html
4. Ative Authentication → E-mail/Senha
5. Crie o Firestore Database em modo teste
6. Cole as regras de segurança no Firestore → Regras
7. Suba o arquivo no GitHub — pronto!
